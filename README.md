# Change-Demo-System

The application is hosted at https://change-demo-system.onrender.com/login

This is a demo change system application used to demostrate the concepts of change management.

There are 2 user roles in the application. User and Admin

User can create, view and edit changes.

Admins can create, approve, view, edit and delete changes as well as creating users, groups and adding users to the groups as well.

The application has a login and register screen. Users can log out at any time once logged in using the log out button located at the top right of the screen.

Changes are moved to the next stage via buttons that are available on the view change screen.

The time of implementation and close will be recorded when a user presses the corresponding implement and close buttons.
